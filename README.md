#  Marketing Funnel & Conversion Performance Analysis  

An end-to-end data analytics project combining **Machine Learning (Python)** and **Power BI** to analyze customer conversion behavior from marketing campaigns. The project focuses on identifying drop-offs, high-performing segments, and key factors influencing conversions.

## 🎯 Objective  
- Analyze marketing funnel performance  
- Identify customer drop-off points  
- Discover high-performing customer segments  
- Understand factors affecting conversion rates  

---

## 📂 Dataset  
- bank-additional-full Dataset  

---

## 🧠 Machine Learning (Python)  

### ✔ Key Steps  
- Data Cleaning & Preprocessing  
- Handling Missing & Categorical Data  
- Exploratory Data Analysis (EDA)  
- Feature Selection  
- Model Building (Classification Models)  

### ✔ Goal  
Predict customer conversion and identify key influencing features  

---

## 📊 Power BI Dashboard  

### 🔹 Features  
- Marketing Funnel Overview  
- Conversion Rate Analysis  
- Drop-off Analysis  
- Customer Segmentation (Job, Education, etc.)  
- Campaign Performance Insights  
- Behavioral Analysis (e.g., call duration impact)  

---

## 🔥 Key Insights  
- High drop-offs observed in early funnel stages  
- Conversion varies across customer segments  
- Certain segments show higher conversion potential  
- Campaign performance can be improved with targeted strategies
